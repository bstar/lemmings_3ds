You may place custom sound files here. Whenever a valid specific
sound file exists in this folder, it is used instead of
corresponding ADLIB.DAT tune. All other sounds and tunes are
still played by ADLIB.DAT.

Format:
Only PCM RIFF WAVE, only 8 or 16 bit, only 1 channel (mono).
Stereo might work, but has not been tested.
